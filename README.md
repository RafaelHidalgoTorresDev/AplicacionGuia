## 🌟 Sobre el Proyecto

**Guía para Mayores** es una aplicación web interactiva diseñada específicamente para ayudar a personas mayores o con dificultades tecnológicas a entender cómo usar servicios digitales del día a día.

Con una interfaz visual, de alto contraste y letras grandes, esta herramienta hace que aprender sea un proceso amigable sin frustraciones.

## 📱 ¿Qué puedes aprender?

Actualmente, la aplicación incluye tutoriales prácticos sobre:
- 💬 **WhatsApp**: Cómo mandar mensajes, fotos y hacer llamadas.
- 🏥 **Centro de Salud**: Cómo pedir citas médicas online.
- 📄 **Certificados**: Cómo descargar documentos importantes de internet.

---

## 🚀 Cómo probarlo tú mismo

1. Descarga el código o clona este repositorio en tu ordenador:
   ```bash
   git clone https://github.com/RafaelHidalgoTorresDev/AplicacionGuia.git
   ```
2. Abre la carpeta del proyecto.
3. Haz doble click sobre el archivo `index.html` para abrirlo en tu navegador favorito.
   *(No se necesita ningún servidor o instalación especial).*

---

## 🎨 Características Técnicas

- **Hecho con amor y código sencillo**: Construido en **HTML5, CSS3 y Vanilla JavaScript**. No depende de frameworks complejos, asegurando la máxima compatibilidad y rapidez.
- **Iconografía amigable**: Uso de íconos claros (Google Material Icons) para guiar al usuario.
- **Tipografía legible**: Fuente 'Roboto' configurada a gran escala, ideal para evitar la vista cansada.
- **Diseño Resposivo**: Se adapta perfectamente a vista de móvil, que es la forma más común en que los mayores intentarán aprender.
